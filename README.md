# How to use:
1) Deploy the bot or just run it by inserting your own BOT token in the specified part
2) Invite the BOT to your server
3) Use on of the commands listed down below to generate random jokes on your wish

# Commands:
### JavaScript
Dark Jokes
```
djoker dark
```
Programming Jokes
```
djoker programming
```
Misc Jokes
```
djoker misc
```
Pun Jokes
```
djoker pun
```
Spooky Jokes
```
djoker spooky
```
Christmas Jokes
```
djoker christmas
```
